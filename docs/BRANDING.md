## Palet Warna

### Warna Utama

```css
--color-primary: #C2410C;       /* Terracotta — CTA, aksen */
--color-primary-dark: #9A3412;  /* Hover gelap, gradient */
--color-secondary: #FB923C;     /* Orange muda — highlight */
--color-dark: #1E293B;          /* Slate — footer, navbar */
--color-cream: #FFF7ED;         /* Background hangat */
--color-text: #0F172A;          /* Teks utama */
--color-text-muted: #64748B;    /* Teks sekunder */
--color-gold: #F59E0B;          /* Rating, badge premium */
--color-white: #FFFFFF;
--color-whatsapp: #25D366;      /* Tombol WhatsApp */
```

### Penggunaan

- **Primary** — Tombol utama, link aktif, badge Best Seller
- **Secondary** — Icon logo, underline nav, border avatar
- **Dark** — Footer, navbar saat scroll, CTA box background
- **Cream** — Background section alternatif
- **Gold** — Bintang rating, statistik hero, badge "New"

## Logo

Icon utama: garpu/utensils (Lucide style) dalam warna secondary, dipasangkan dengan wordmark "Rasa Nusantara" font Poppins Bold.

File: `assets/favicon.svg`

## Tone of Voice

- Bahasa Indonesia profesional dan hangat
- Mengundang tanpa berlebihan
- Menekankan kualitas, tradisi, dan sentuhan modern
- Hindari slang berlebihan

## Aplikasi Visual

- Gradient orange pada tombol primary dan hero overlay
- Glassmorphism ringan pada stat hero dan navbar scrolled
- Rounded corners konsisten (8px–20px)
- Shadow lembut untuk depth
- Foto kuliner dengan warm lighting cinematic
