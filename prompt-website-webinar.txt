==================================================
JAVASCRIPT — FITUR WAJIB
==================================================

File js/main.js wajib implementasi (vanilla JS, IIFE, 'use strict'):

1. Navbar scroll
   - Tambah class .scrolled saat window.scrollY > 50

2. Mobile menu
   - Toggle .open pada menu & hamburger
   - aria-expanded true/false
   - body overflow hidden saat menu buka
   - Tutup saat klik link, klik luar, tombol Escape

3. Smooth scroll
   - Semua a[href^="#"] → scroll dengan offset navbar
   - Tutup mobile menu setelah klik anchor

4. Active nav link
   - Highlight .navbar__link.active sesuai section yang terlihat

5. Scroll animations
   - IntersectionObserver untuk .fade-up → .visible
   - Fallback: langsung .visible jika Observer tidak support

6. Testimoni slider
   - Prev / Next button
   - Dot indicator dinamis (dibuild via JS)
   - Auto-slide interval 5000ms
   - aria-label pada tombol navigasi

Jangan gunakan jQuery, React, Vue, atau library lain.
